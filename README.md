# WJCT-child-theme
# This is a child theme of Twenty Twenty-Five by WordPress, built for a WJCT WordPress Developer skills test

## What this child theme does?
- alters the background color of the site to Alice Blue
- changes headline front to Pallete Mozaic
- changes body font to Oswald
- adds a custom fuction that includes "Built for WJCT skills test" message at the bottom of the footer

## How to install
1. Download or clone this repository into your WordPress themes directory at wp-contet/themes
2. Ensure that Twenty Twenty-Five parent theme is installed
3. Access your WordPress admin and navigate to Themes under the Appearnace section
4. Activate WJCT Skill Test

## How to test
- Visit the site with the activated child theme and verify if the background color is Alice Blue
- Check that the headings on the site display with the Pallete Mozaic
Check that the body text is rendering with font Oswald
- Scroll to the bottom of the page to check if the added footer text is displaying in the bottom left corener of the page

## What I would improve
Given more time I would research and target more specific attributes, tags, and classes to ensure that no unintended styling outcomes could happen. I would also create font fallbacks in case the google fonts fail to load. Alternatively I would host the fonts locally in the theme. This approach would be safer since it does not rely on a third party.
